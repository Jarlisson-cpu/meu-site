Aplicativo de Acessibilidade da Orla da Atalaia

Esta é uma página de teste para o aplicativo de acessibilidade que mostra locais com recursos de acessibilidade na Orla da Atalaia em Aracaju.

Acessar o Aplicativo

Instruções para Teste Local

  1. Para testar todas as funcionalidades do aplicativo, incluindo o mapa e recursos offline, é recomendado usar um servidor local.
  2. Se você tem Python instalado, pode iniciar um servidor local com o comando:
    python -m http.server 8000 (Python 3)
    ou
    python -m SimpleHTTPServer 8000 (Python 2)
  3. Depois, acesse http://localhost:8000 no seu navegador.
  4. Para testar os recursos de acessibilidade, experimente os botões no topo da página (alto contraste, tamanho da fonte, narração).
  5. Teste os filtros para ver diferentes tipos de locais acessíveis.
  6. Clique nos marcadores do mapa ou nos cards para ver detalhes dos locais.

Nota: Para uma experiência completa, é recomendado criar os ícones conforme as instruções no arquivo README.txt na pasta "icons".
