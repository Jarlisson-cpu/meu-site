INSTRUÇÕES PARA ÍCONES DO APLICATIVO

Para que o aplicativo funcione corretamente como um Progressive Web App (PWA), você precisará criar os seguintes ícones:

1. icon-72x72.png (72x72 pixels)
2. icon-96x96.png (96x96 pixels)
3. icon-128x128.png (128x128 pixels)
4. icon-144x144.png (144x144 pixels)
5. icon-152x152.png (152x152 pixels)
6. icon-192x192.png (192x192 pixels)
7. icon-384x384.png (384x384 pixels)
8. icon-512x512.png (512x512 pixels)
9. map-shortcut.png (96x96 pixels) - Ícone para atalho do mapa
10. list-shortcut.png (96x96 pixels) - Ícone para atalho da lista de locais

Recomendações:
- Use o símbolo internacional de acessibilidade como base para o ícone
- Mantenha um design simples e reconhecível mesmo em tamanhos pequenos
- Use cores contrastantes para melhor visibilidade
- Salve todos os ícones no formato PNG com fundo transparente

Você pode criar estes ícones usando ferramentas como:
- Adobe Illustrator
- Adobe Photoshop
- GIMP (gratuito)
- Inkscape (gratuito)
- Canva (online)
- Figma (online)

Após criar os ícones, substitua este arquivo README.txt pelos arquivos de imagem correspondentes.
