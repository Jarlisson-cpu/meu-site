INSTRUÇÕES PARA CAPTURAS DE TELA DO APLICATIVO

Para melhorar a experiência de instalação do Progressive Web App (PWA), você precisará criar as seguintes capturas de tela:

1. screen1.png (1280x720 pixels) - Captura de tela mostrando o mapa com os pontos de acessibilidade
2. screen2.png (1280x720 pixels) - Captura de tela mostrando a lista de locais acessíveis

Recomendações:
- Capture telas que mostrem claramente as principais funcionalidades do aplicativo
- Certifique-se de que as capturas estejam nítidas e bem enquadradas
- Inclua exemplos de pontos de acessibilidade no mapa
- Mostre a interface com o modo de alto contraste ativado em uma das capturas para destacar os recursos de acessibilidade

Como obter as capturas:
1. Abra o aplicativo em um navegador
2. Redimensione a janela para 1280x720 pixels
3. Use a ferramenta de captura de tela do seu sistema operacional ou uma extensão do navegador
4. Salve as imagens como PNG
5. Nomeie-as como "screen1.png" e "screen2.png"
6. Coloque-as nesta pasta (screenshots)

Após criar as capturas de tela, substitua este arquivo README.txt pelos arquivos de imagem correspondentes.
